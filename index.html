JORNADA DE PROGRAMAÇÃO

Este jogo é totalmente gratuito e educacional voltado para crianças do ensino fundamental, projetado para ensinar os fundamentos da programação de forma divertida e envolvente

Para jogar nosso jogo, é preciso apenas apertar no botão abaixo, onde vc vai ser redirecionado ao jogo, e clicar em "run game". Com isso, voce ja terá acesso a 1º versão do nosso jogo para aprimorar seus conhecimentos e/ou utiliza-ló para métodos de ensino!!

Bom, o jogo é baseado em fases, onde cada fase terá perguntas sobre a linguagem de programação C++, e para passar de fase é necessário realizar os códigos pedidos na tela, BOA SORTE!!
